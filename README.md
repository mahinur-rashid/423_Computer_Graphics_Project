# Sky Dominator (OpenGL)

Sky Dominator is a two-player game developed using OpenGL for CSE423: Computer Graphics course project. The game features two spaceships engaged in a shootout while dodging falling obstacles.\
The rendering is achieved using custom-written midpoint line and circle drawing algorithms solely using GL_POINTS from OpenGL.

- Two-player gameplay with spaceships.
- Bullet shooting and dodging falling obstacles.
- Simple interface and game mechanics.

## Installation (Python Interpreter Required)
1. Clone the repository:
    ```sh
    git clone https://github.com/mahinur-rashid/423_Computer_Graphics_Project.git
    ```
2. Navigate to the project directory:
    ```sh
    cd 423_Computer_Graphics_Project
    ```
3. Run the game:
    ```sh
    python Game.py
    ```

## Gameplay
In Rocket Shootout, two players control their respective spaceships, shoot bullets at each other, and avoid falling obstacles. The game ends when one player's health reaches zero.

## Controls
### Player 1:
- **W**: Move up
- **S**: Move down
- **A**: Move left
- **D**: Move right
- **SPACE**: Shoot

### Player 2:
- **I**: Move up
- **K**: Move down
- **J**: Move left
- **L**: Move right
- **P**: Shoot

### On-screen Buttons:
- **Left-most button icon**: Restart
- **Middle button icon**: Play/Pause
- **Right-most button icon**: Exit

### Keyboard:
- **ESC key**: Play/Pause
